## To-do list app

:fork_and_knife: from <https://github.com/nashville-software-school/ng-todo-boilerplate>

### Run locally

```bash
git clone git@github.com:kenziebottoms/nss-front-05-angular-todo.git
cd nss-front-05-angular-todo/lib
npm install
grunt
```
